Bibliothèque de logiciels Signum

Le logiciel Signum est open source.

Veuillez consulter le guide intitulé « Premiers pas avec Signum » avant d’installer un logiciel Signum.

Choisir BTDEX pour un progiciel tout-en-un facile à utiliser qui comprend les fonctions essentielles du portefeuille, le logiciel de traçage, le logiciel de minage et l’échange décentralisé de Signum.

Logiciel de portefeuille
Signum Node
  • Le logiciel de référence contenant tous les protocoles. Comprend l’interface portefeuille classique ainsi qu’un client léger.
  • Requis pour l’exploitation d’un nœud synchronisé.
  • Plateforme: Windows, macOS, Linux, iOS, Android
  • https://signum.community/signum-node-installation/
Phoenix
  • La version autonome de l’application light node incluse avec Signum Node.
  • Fait référence à une copie de la blockchain stockée sur les nœuds du réseau plutôt qu’à sa propre copie. Pour faire fonctionner un nœud complet, utilisez Signum Node.
  • Plateforme: Windows, macOS, Linux, iOS, Android
  • https://github.com/signum-network/phoenix/releases/
Mobile
  • Client léger pratique pour téléphones mobiles et autres appareils Android.
  • GooglePlayStore: https://play.google.com/store/apps/details?id=com.signum.phoenix
Logiciel de traçage
Signum Plotter (version compilée à venir: veuillez utiliser Engraver ci-dessous)
  • Le traceur le plus rapide de Signum, dispose d’entrées / sorties directes et asynchrones, d’un support SIMD (sse2, avx, avx2, avx512f) et d’un support GPU.
  • Platform: 64bit only!
  • https://github.com/signum-network/signum-plotter/releases/
Instructions pour le graveur

Instructions de base pour Windows:

  • Créez un dossier vide pour contenir vos fichiers de tracé sur votre système local ou sur un disque externe. N’utilisez pas d’espaces. Exemple: E: \ PlotFiles ou E: \ Plot_Files
  • Téléchargez et décompressez la dernière version de votre système d’exploitation. Ouvrez le dossier décompressé et double-cliquez sur l’application (SignumPlotterGUI). Suivez les invites de votre système pour installer.
  • Dans l’application, saisissez l’identifiant numérique du compte, situé dans le coin supérieur gauche de votre portefeuille. (Remarque: ce n’est pas l’adresse du compte qui commence par « S »)
  • Dossier de sortie: accédez au dossier que vous avez créé pour contenir vos fichiers de tracé.
  • Démarrer Nonce: utilisez la valeur par défaut de zéro si vous tracez pour la première fois. Sinon, cliquez sur «auto à partir du dernier fichier» et accédez au dernier fichier de tracé que vous avez créé afin que le traceur puisse déterminer automatiquement le nonce de départ pour une nouvelle session de traçage.
  • Taille à tracer: choisissez «maximum» pour remplir l’espace disponible de votre disque avec des fichiers de tracé individuels d’une taille spécifique. Choisissez « valeur » pour créer un seul fichier d’une taille particulière. Chaque nonce utilisera 256 Ko d’espace disque. Les performances ne sont pas affectées par la taille du fichier choisi. Cependant, diviser vos fichiers de tracé en plusieurs fichiers plus petits peut être bénéfique si vous devez les déplacer à un moment donné ou si vous devez en supprimer une partie pour libérer de l’espace à une autre fin plus tard.
  • Cliquez sur Démarrer le traçage. Laissez la session de traçage se terminer. Selon le nombre de tracés que vous créez, ce processus peut prendre des heures ou des jours. Il peut être interrompu en cliquant sur « Arrêter le traçage » et repris plus tard.

Remarques:

  • Pour choisir entre l’utilisation de votre CPU ou GPU et pour définir des limites sur la mémoire et les cœurs utilisés, effectuez une sélection dans l’onglet Paramètres avancés.
  • Évitez les fichiers de tracé qui se chevauchent (doublons). Considérez attentivement vos numéros nonce de départ. Les fichiers de tracé en double n’ont aucune valeur.
Graveur

Logiciel d'exploitation minière
Signum Miner
  • Un mineur écrit en Rust. Il comprend direct io, avx512f, avx2, avx, sse et OpenCL
  • Soumet uniquement le meilleur délai à chaque tour, ce qui réduit le trafic réseau et le stress sur les opérations du pool. (Soumettre des délais de plus en plus petits au fur et à mesure qu’un tour progresse n’est plus pertinent après la fourche Sodium)
  • Développeur principal: Consortium PoC
  • https://signum.community/miner-installation-guide/
  • Plateforme: Windows, Linux
Scavenger
  • Un mineur écrit en Rust. Il comprend direct io, avx512f, avx2, avx, sse et OpenCL
  • Développeur principal: Consortium PoC
  • https://signum.community/miner-installation-guide/
  • Plateforme: Windows, macOS, Linux, Unix (64 bits)

Le fichier de configuration utilisé par Scavenger et créé par Gyra est un fichier .yaml. Les utilisateurs peuvent le télécharger dans leur navigateur et l’enregistrer dans le répertoire racine où se trouve l’exécutable Scavenger.

  • Développeur principal: feyd27

SignumJS - SDK Javascript pour Signum

Un SDK moderne écrit en Typescript fournissant des fonctionnalités communes pour browsers et nodejs pour interagir avec la blockchain Signum Network.

  • @signumjs/core ( Le package principal fournissant une API étendue pour l’interaction blockchain)
  • @signumjs/contrats ( fonctions pour contrats intelligents)
  • @signumjs/crypto ( fonctions cryptographiques)
  • @signumjs/util ( conversion commune et autres fonctions utiles)
  • @signumjs/http ( simplifié Couche HTTP, avec des types de réponse cohérents et une gestion des exceptions)
  • @signumjs/moniteur ( une classe pour exécuter des opérations asynchrones récurrentes avec la fonction de dé/sérialisation, idéale pour écouter les transactions blockchain)
  • Développeur principal : blankey1337, Ohager
  • Relier: https://github.com/signum-network/signumjs
  • Documentation: https://signum-network.github.io/signumjs/

Guide:

Cet outil peut être utilisé pour générer un nouveau portefeuille ou avec un portefeuille existant.

Les portefeuilles papier peuvent être utilisés pour transférer des fonds à l’aide de clés publiques ou comme document de référence pour le stockage à froid (les fonds sont toujours sur la blockchain, mais le document papier fournit les informations d’accès requises).

Pour utiliser l’outil, remplissez tous les champs requis, puis cliquez sur le bouton souhaité. Le portefeuille papier sortira sous forme de fichier .pdf imprimable.

Générateur d'adresses de vanité

Le générateur d’adresses personnalisées est un outil permettant de créer des adresses de compte Signum personnalisées ou personnalisées. Toutes les adresses doivent avoir « Rafale » pour les cinq premiers caractères de tête. The capital letter « i » (I), the capital letter « o » (O), zero (0), and the number one (1) are not permitted. To generate a vanity address, enter characters into the form provided. Set the resources that should be used (number of CPU threads) and desired length of the passphrase.

L’algorithme qui recherche le compte avec la chaîne souhaitée est un algorithme de force brute. Il vérifie chaque compte Signum et combinaison de phrase secrète de manière séquentielle. The time it takes to find a particular string increases significantly with every additional character. A string of 5 characters at the end takes approximately 4 minutes (more or less depending on computer resources). Each additional character increases the time required exponentially. Une chaîne de 8 caractères prendrait environ une heure. Longer strings could take weeks.
